To run:
	Double-click "Attention Meter.exe" or run from command line.

To run in server mode (requires netcat):
	Double-click "attention meter remote daemon.cmd" or run from command line.

To connect a client to server (requires netcat):
	Type "nc HOSTNAME 5001" where HOSTNAME is the hostname of server.

To exit:
	Select any window and press space bar.

For more detailed information, see the PDF manual in this directory.
